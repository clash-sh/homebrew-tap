# Homebrew Clash

Homebrew tap for [Clash](https://github.com/clash-sh/clash) - Manage merge conflicts across git worktrees for parallel AI coding agents.

## Installation

```bash
brew tap clash-sh/clash
brew install clash
```

## Usage

After installation, you can use the `clash` command:

```bash
clash status
clash watch
```

For more information, see the [main repository](https://github.com/clash-sh/clash).
